1. Clone repository:
$ git clone [repository-url]
$ cd laravel-12

2. Install dependencies:
$ composer install
$ npm install

3. Copy file .env.example:
$ cp .env.example .env

4. Generate application key:
$ php artisan key:generate

5. Setup database:
$ php artisan migrate
$ php artisan db:seed

6. Compile assets:
$ npm run build

7. Jalankan server:
$ php artisan serve
